# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-12-23
### Added
- **Core Engine**: Initial release of the Foveated Streaming Engine (WASM).
- **Compression**: Implemented Dynamic Region of Interest (ROI) diffing.
- **Rendering**: Hardware-accelerated canvas blurring and composition.
- **Universal Wrapper**: `index.js` wrapper for easy browser integration.
- **Documentation**: Comprehensive README and API Reference.
